- **Warrior** — Guardian (tank), Berserker (DPS), Paladin (hybrid)
- **Mage** — Pyromancer (fire DPS), Frostbinder (control), Arcanist (burst)
- **Ranger** — Sharpshooter (precision), Beastmaster (pets), Shadowstalker (stealth)
- **Artisan** — Blacksmith (weapons/armor), Alchemist (potions), Enchanter (magic items)
