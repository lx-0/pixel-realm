## Game Description

**PixelRealm** is a browser-based pixelated MMORPG where the world never stops evolving. Explore biome-rich zones, battle monsters, craft gear, and form guilds — all in a living world powered by AI-generated quests, NPC dialogue, and dynamic world events. No two sessions are alike.

Built in the tradition of SNES-era RPGs with a modern twist, PixelRealm features 16×16 pixel art across 13+ hand-designed zones spanning lush forests, scorching deserts, frozen peaks, haunted bogs, and floating sky islands. Fight in real-time combat with skills and cooldowns, team up with friends for dungeon boss raids, or challenge rivals in the PvP arena. Between adventures, craft powerful gear, decorate your player house, and trade on the open marketplace.

Whether you're a solo explorer chasing LLM-generated storylines or a guild leader commanding territory wars, PixelRealm has something for you. Jump in — all you need is a browser.

## Features

- **Massive Pixel World** — 13+ unique zones across 5 biome tiers, from the tutorial Verdant Hollow to the endgame Void Rift and Sunken Titan's Trench
- **Real-Time Combat** — Semi-real-time skill-based combat with hotbar abilities, cooldowns, dodge rolls, and combo bonuses
- **AI-Generated Quests** — Server-side LLM creates personalized quest lines based on your choices, reputation, and world state — every player's story is different
- **Dynamic World Events** — AI-triggered invasions, treasure hunts, faction wars, and seasonal story arcs that reshape the global map
- **Multiplayer** — See other players in the world, chat, trade, and team up for dungeon raids and boss fights
- **PvP Arena** — Ranked 1v1 gladiator combat with spectators and leaderboards
- **Deep Crafting System** — Combine drops and resources at crafting stations to forge gear, brew potions, and build tools
- **Player Housing** — Own land plots, build and furnish your home, set up shops and crafting hubs
- **Guild System** — Form guilds, claim territory, run guild raids, and wage guild-vs-guild battles
- **4 Character Classes** — Warrior, Mage, Ranger, and Artisan, each with 3 branching skill tree archetypes
- **Day/Night Cycle** — Palette-swapped lighting that shifts from warm dusk to cool moonlit nights
- **NPC Reputation** — Faction standings with Merchants, Mages Guild, Bandit Clans, and more alter NPC behavior and unlock unique content
- **Seasonal Content** — New story arcs, classes, and biomes every season keep the world fresh

## Controls

### Keyboard & Mouse

| Action | Key / Input |
|---|---|
| Move | WASD or Arrow Keys |
| Click-to-Move | Right-click |
| Basic Attack | Spacebar or Left-click enemy |
| Skills 1–6 | Number keys 1–6 or click hotbar |
| Interact / Talk | F or Left-click NPC |
| Inventory | I |
| Map | M |
| Quest Log | J |
| Chat | Enter |
| Sprint | Hold Shift |
| Dodge / Roll | Q |
| Target Nearest | Tab |

### Touch Controls (Mobile)

| Action | Gesture |
|---|---|
| Move | Virtual D-pad |
| Attack | Tap enemy |
| Skills | Tap hotbar icons |
| Interact | Tap NPC |
| Sprint | Double-tap D-pad |
| Dodge | Swipe |
| Menus | Tap bag / map / quest icons |

## System Requirements

PixelRealm runs entirely in your browser — no download or install needed.

| | Minimum | Recommended |
|---|---|---|
| **Browser** | Chrome 90+, Firefox 90+, Safari 15+, Edge 90+ | Latest Chrome or Firefox |
| **Graphics** | WebGL 1.0 support | WebGL 2.0 support |
| **RAM** | 2 GB available | 4 GB available |
| **Internet** | Broadband connection (multiplayer) | Stable broadband, <100ms ping |
| **Display** | 960×540 or larger | 1280×720 or larger |
| **Input** | Keyboard + mouse | Keyboard + mouse (touch supported) |

*Runs on Windows, macOS, Linux, and Chromebook. Mobile browser support is experimental.*
